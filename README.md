## ✨ Features  
- 🔐 User Authentication (Signup / Login)  
- 📝 Create, Edit & Delete Travel Blogs  
- 📂 Upload Images with Blogs  
- ❤️ Like & Comment System  
- 🎨 Responsive & Modern UI  
- ⚡ REST APIs with secure backend
